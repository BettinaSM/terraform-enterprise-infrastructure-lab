# Terraform Enterprise Infrastructure Lab

## Overview

This project simulates enterprise-grade infrastructure provisioning using Terraform across AWS and GCP environments.

The lab focuses on:
- Infrastructure as Code (IaC)
- Cloud security best practices
- IAM governance
- Network segmentation
- Security controls
- Multi-cloud architecture concepts

- Infrastructure as Code (IaC)
