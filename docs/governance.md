# Governance Concepts

This project demonstrates foundational governance and Policy as Code concepts.

## Implemented Concepts

- Infrastructure governance
- Secure-by-default configurations
- Policy validation
- Cloud security controls
- DevSecOps automation

## Policy Examples

- Prevent public SSH access
- Restrict insecure cloud storage
- Simulate enterprise security guardrails

## Technologies

- Terraform
- Open Policy Agent (OPA)
- Rego
- GitHub Actions
